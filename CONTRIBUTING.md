# Contributing

Thank you for your interest in improving the NHS Retirement Planner.

This project is created and maintained by **Assegai Studios**.

## Ways to contribute

You can help by:

- Reporting formula errors
- Suggesting clearer assumptions
- Improving documentation
- Flagging outdated pension, tax, or State Pension assumptions
- Suggesting usability improvements
- Testing retirement scenarios and reporting unexpected results

## Before contributing

Please remember that this workbook is not financial advice. Contributions should focus on accuracy, clarity, usability, documentation, and transparency.

Do not submit personal financial information, pension statements, National Insurance numbers, salary details, or any other sensitive data.

## Issues

When opening an issue, please include:

- The workbook version
- The sheet name
- The cell or formula affected, if relevant
- A clear description of the issue
- The expected result
- The actual result

## Pull requests

Pull requests are welcome. Please keep changes focused and explain the reason for the change.

By contributing, you agree that your contribution may be included in the project under the repository licence.
